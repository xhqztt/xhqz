# 考研打卡 PWA · 安装与部署指南

## 文件清单
- `index.html` — 主程序
- `manifest.json` — PWA 应用清单
- `sw.js` — Service Worker（离线缓存）
- `icon-192.png` / `icon-512.png` — 应用图标
- `apple-touch-icon.png` — iOS 专用图标

---

## 三种使用方式（任选其一）

### 方式 A：本地浏览器使用（最简单，电脑端）
直接双击 `index.html` 用浏览器打开即可。
⚠️ 注意：本地文件打开时，Service Worker 离线缓存不会生效，但所有其他功能正常。

### 方式 B：安装到手机主屏幕（推荐 ⭐⭐⭐）
需要先把整个 `kaoyan-app` 文件夹放到一个网址上访问。最简单的免费方案：

#### 用 GitHub Pages 部署（5分钟搞定）
1. 注册 [GitHub](https://github.com) 账号（已有跳过）
2. 新建仓库（New repository），名字随意，例如 `kaoyan`，**设为 Public**
3. 上传整个 `kaoyan-app` 文件夹内的所有文件到仓库
4. 进入仓库 Settings → Pages → 选 Source: `main` 分支，保存
5. 等 1-2 分钟，会得到一个网址，类似 `https://你的用户名.github.io/kaoyan/`
6. 在手机浏览器打开这个网址：
   - **iPhone**：用 Safari 打开 → 点底部分享按钮 → "添加到主屏幕"
   - **Android**：用 Chrome 打开 → 浏览器会弹出"添加到主屏幕"提示，或菜单里手动选
7. 主屏幕会出现一个图标，点开就是全屏 App，可离线使用

#### 用 Vercel 部署（更快）
1. 注册 [Vercel](https://vercel.com)（可用 GitHub 账号一键登录）
2. New Project → Upload → 拖入整个 `kaoyan-app` 文件夹
3. 自动部署后得到一个网址，按上面方式 6-7 添加到主屏幕

### 方式 C：直接把文件传到手机本地（次推荐）
1. 把整个 `kaoyan-app` 文件夹通过微信文件传输助手 / AirDrop / 网盘等方式传到手机
2. 用手机文件管理器找到 `index.html`，用浏览器打开
3. 部分手机可以直接"添加到主屏幕"
4. ⚠️ 这种方式部分功能（如离线缓存）可能受限，建议优先用方式 B

## 数据安全提醒

- 所有打卡、番茄记录、设置都存在**你设备的浏览器本地**，不会上传到任何服务器
- 数据归属于当前浏览器：清缓存、换浏览器、换设备都会丢失数据
- **建议每周用番茄钟页面底部的"导出备份"按钮导出一次 JSON 文件**

## 常见问题

**Q: PWA 装好后没有网络能用吗？**
A: 可以。第一次访问后所有文件会被 Service Worker 缓存，之后离线打开依然正常运行。

**Q: 我在手机和电脑上的数据能同步吗？**
A: 不能自动同步，因为数据存在浏览器本地。要同步只能手动导出 JSON 文件再在另一台设备导入。

**Q: 我能修改任务内容吗？**
A: 当前版本任务是预设的，要修改需要编辑 index.html 文件中的 buildDay 函数。如果你想要可自定义任务，告诉我可以加这个功能。

**Q: 数据存在哪里？**
A: 浏览器 localStorage，键名以 `kaoyan_` 开头。
